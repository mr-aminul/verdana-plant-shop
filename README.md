# 🌿 Verdana — Premium Indoor Plant Delivery

A fully functional, responsive landing page for a fictional premium indoor plant delivery brand. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

<img width="1920" height="4157" alt="image" src="https://github.com/user-attachments/assets/5988ec52-4d13-49a5-bd15-88798a90c15a" />


## ✨ Features

- **Product catalog** with 6 curated plants and detailed modals
- **Category filtering** — Easy care, Low light, Air purifying, Statement
- **Shopping cart** — slide-out drawer with quantity controls and live totals
- **Checkout flow** — form validation, order summary, and success screen
- **Newsletter signup** — with email validation and discount confirmation
- **Toast notifications** — lightweight feedback for all interactions
- **Fully responsive** — works on mobile, tablet, and desktop

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (no libraries)
- Google Fonts — Playfair Display + Inter

## 🚀 Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/verdana-plant-shop.git
cd verdana-plant-shop
```

Then just open `index.html` in your browser. No build step, no installs.

Or serve it locally:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

## 🎨 Design Decisions

- **Color palette** — Deep forest green (`#1a3a2a`) paired with warm cream (`#f7f3eb`) for a premium organic feel
- **Typography** — Playfair Display for headings (editorial, high-end), Inter for body (clean, readable)
- **No frameworks** — kept intentionally lightweight; the whole site is a single HTML file
- **BDT pricing** — localised for the Bangladeshi market (৳)
